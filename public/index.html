Features

Everything You Need for Compliance

Purpose-built to handle Vermont's unique liquor transaction requirements so you can focus on running your business.

Automatic Cart Splitting

Mixed carts are automatically separated into liquor and retail items. No manual sorting required — the API handles the split at checkout.

Dual Transaction Processing

Two independent Stripe charges per order — one for retail (6% VT sales tax) and one for liquor (10% VT liquor tax) — exactly as the state requires.

Automatic Refund Protection

If one half of a split transaction fails, the successful charge is automatically refunded. Your customers are never partially billed.

Tax Reporting Ready

Tax amounts are embedded in every Stripe charge's metadata, making Vermont state remittance reporting straightforward and audit-proof.

PostgreSQL Audit Log

Every transaction — successful, failed, or refunded — is logged to PostgreSQL with full details for regulatory compliance and internal audits.

Secure API Key Auth

Each retailer gets a unique API key tied to their account. Every request is validated before processing — no unauthorized access.

How It Works

Simple Integration, Full Compliance

Integrate once and let the API handle Vermont's split-transaction requirements automatically.

1

Send Your Cart

POST your cart items to our API. Each item includes a name, category (liquor or retail), price, and quantity.

2

Automatic Splitting

The API separates liquor items from retail goods and calculates the correct Vermont tax rate for each group.

3

Dual Stripe Charges

Two independent payment intents are created through Stripe — one for retail at 6% and one for liquor at 10% — per Vermont law.

4

Audit & Reporting

Every transaction is logged with full metadata for tax remittance and compliance audits. You get a clean JSON response with both charge IDs.
